# Go SemVer Demo Project

Простой веб-сервер на Go для демонстрации правил семантического версионирования (SemVer).

## Система версионирования

Этот проект следует правилам [Semantic Versioning 2.0.0](https://semver.org/):
- **MAJOR.MINOR.PATCH** (например, 1.2.1)
- **MAJOR**: Несовместимые изменения API.
- **MINOR**: Новые функции, совместимые с предыдущими версиями.
- **PATCH**: Исправления ошибок, совместимые с предыдущими версиями.


```bash
go run main.go
```

Сервер запустится на порту 8080
